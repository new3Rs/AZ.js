Class: AZjsEngineBase

AZjsEngineBase(size, komi, evaluatePlugin)

対局を行う思考エンジンの基本クラスです。 ウェブワーカで動かすことを前提に、メインスレッドのアプリとNeuralNetworkの2つと通信しながらMCTSを実行します。 AZjsEngineという拡張クラスを作成して使います。
